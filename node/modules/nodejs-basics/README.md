# Module "Node.js basics"

## In this module, students study:

- Node.js APIs:
  - Filesystem API
  - Streams API
  - Modules API
  - Timers API
  - CLI API
  - Child processes API
  - Worker Threads API
  - Operation System API
  - Zlib API
  - Crypto API
- Event Loop
- Blocking vs Non-blocking I/O
